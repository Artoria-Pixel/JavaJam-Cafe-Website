JavaJam Coffee House Report

The JavaJam website uses hierarchical site organization. This is the most appropriate as the site has only four pages. Using hierarchical organization allows for easy navagation and categorization of content.

Three design practices that have been well implemented for this site:

  • Consistent site header/logo.
  • Good contrast between text and background.
  • Color is used consistently.

Three design practices that could be implemented in a better way:
  • Slightly larger footer text for better visibility.
  • More graphics depicting JavaJam products.
  • Use of a more appealing color scheme.

How else would you improve the website?
  • I would add more images of JavaJam products on the menu page.
  • I would have a link to the musician's website or contact information on the music page.
  • I would add song clips or a track list to the music page.
